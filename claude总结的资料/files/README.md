## 这份资源库怎么用

**两种用法：**

1. **AI 调用**：用户聊到某个职业方向，AI 根据"用户场景"调用对应资源块。比如用户提到"想做电工"，AI 自动给出 Skilled Trades Ontario 链接 + 学徒搜索网站 + 政府补贴信息。

2. **你审视**：你可以快速看一眼自己产品的"知识储备"，哪些模块充实、哪些模块缺。

## 模块 1：求职平台（最完整）

**核心思路**：不按"工具类型"分，而按"用户场景"分。

### 通用 - 所有人都该用
- **Job Bank Canada** — 政府官方，必看
- **LinkedIn** — 白领必备
- **Indeed Canada** — 岗位最多
- **Glassdoor** — 看公司内幕

### 按职业类型分
- **财务会计** → Robert Half、CPA Canada Job Board
- **IT 科技** → LinkedIn、Dice、Indeed
- **教育** → Education Canada、学校官网
- **医疗** → MedHunters、各省卫生局
- **技工** → Job Bank、Apprenticesearch、ZipRecruiter
- **餐饮** → Indeed、店家直接问
- **NGO** → Charity Village
- **法语区** → Jobillico
- **学生** → TalentEgg

**亮点**：每个平台都标注了"强项"、"适合谁"、"提示"——AI 推荐时不只是给链接，还会给使用建议。

## 模块 2：新移民免费就业服务（最被忽视）

这个模块价值巨大但 90% 新移民不知道。**全是政府资助、完全免费、专门给 PR 用的。**

### 安省（GTA）—— 6 个核心机构
1. **JVS Toronto** — 1对1咨询、TRIEC 导师匹配
2. **ACCES Employment** — Speed Mentoring（一晚见 10 个行业内人）
3. **YMCA of Greater Toronto** — 带薪实习
4. **Skills for Change** — Bridge Training
5. **TNO Toronto** — 免费 Coding 培训
6. **TRIEC** — GTA 雇主网络中心

### BC 省 —— 3 个
- ISSofBC、MOSAIC BC、**S.U.C.C.E.S.S.**（华人友好，有中文服务）

### 阿省 / 魁省 / 全国 —— 各 1-2 个

**对你产品的意义**：
当用户聊到"找不到对口工作、缺本地经验"时，AI 应该立刻推荐这些免费资源——**而不是推荐培训学校（要花钱）或者私人咨询（要花钱）**。

这是真正的"服务用户"。

## 模块 3：学历职业认证（必经流程）

**新移民来加拿大第一件事**——做学历认证。

### 综合学历认证
- **WES Canada** — 最主流，$220-280 CAD
- **ICAS Canada** — WES 处理不了的特殊情况
- **IQAS** — 阿省专用

### 各省技工认证
- 安省 → **Skilled Trades Ontario**（140+ 种技工）
- BC → SkilledTradesBC
- 阿省 → Tradesecrets Alberta

### 专业资质认证
- 护士 → **NNAS**（全国第一步）→ CNO（安省）/ BCCNM（BC）
- 会计 → **CPA Canada**
- 工程师 → PEO（提示：周期长，要慎重）
- 教师 → OCT（安省）

**对你产品的意义**：
用户提到"我国内是 XX 职业"时，AI 必须知道**这个职业在加拿大需要什么认证**——这是基础信息。资源库已经覆盖最常见的 6 大类。

## 模块 4：工资数据（讲价值时必用）

**做职业规划必看的官方数据来源：**

- **Job Bank Wages** — 按 NOC 代码查工资
- **Statistics Canada** — 宏观行业数据
- **NOC 数据库** — 职业代码和定义

AI 给推荐时必须引用这三个之一作为数据来源——这就是你之前说的"信息来源权威"的体现。

## 模块 6：培训学校（特殊处理）

**⚠️ 这个模块我故意做得保守。**

理由：培训学校鱼龙混杂，私立机构很多是收钱推荐套路。

**v1 只列公立学院**（这些有政府认证，质量有保障）：
- 安省：George Brown、Centennial、Seneca、Humber、Algonquin
- BC：BCIT、VCC

**私立机构怎么处理？**
我设了硬规则：进入推荐池前必须满足
1. 有公立认证或政府认可
2. 有真实毕业生案例可查
3. 收费透明无套路
4. **不付费给我们做推荐**

**这条第 4 条最重要**——以后如果有培训机构想付费上推荐位，我们的产品定位是"要他们的真实数据，不要他们的钱"。这才是你跟竞品（房大师、各种推荐网站）的差异化。

## 跟案例库的关系

| | 案例库 | 资源库 |
|---|--------|--------|
| **内容** | 真实人物的真实路径 | 官方机构、平台、政策链接 |
| **价值** | "情感共鸣 + 路径参考" | "下一步具体怎么做" |
| **来源** | 你挖（小红书等） | 我建（官方网站） |
| **AI 用法** | 总结时匹配相似案例 | 给完建议后附上对应资源 |

**两个库一起用，AI 的回复才完整：**

```
1. 推荐方向（基于用户对话）
2. 匹配案例（"有人跟你一样走过这条路"）—— 来自案例库
3. 具体资源（"这是政府认证机构、求职平台、免费服务"）—— 来自资源库
4. 下一步建议
```
