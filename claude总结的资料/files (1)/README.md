## 🎯 现在你该做什么

### 第 2 步：给 Claude Code 集成

打开 Claude Code，做两件事：

1. **上传 4 个文件**：
   - cases.json
   - tactics.json
   - resources.json
   - v2_design_doc.md

2. **复制 `CLAUDE_CODE_INSTRUCTIONS.md` 里的「集成指令」整段**给它

Claude Code 会按 11 个步骤集成 V2。

### 第 3 步：配置 Railway 环境变量

集成完成后，Claude Code 会告诉你需要哪些环境变量。

你去 Railway 控制台 → Variables 添加：
- `ANTHROPIC_API_KEY`
- `DATABASE_URL`（Postgres 自动）
- `RESEND_API_KEY`
- `ADMIN_PASSWORD`
- `WECHAT_CONTACT`

Railway 会自动重新部署。
